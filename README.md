# Prova Técnica - API IBGE

Projeto desenvolvido em Python para:

* leitura de CSV;
* consumo da API pública do IBGE;
* tratamento e normalização de municípios;
* geração de resultado.csv;
* cálculo de estatísticas;
* envio dos resultados para API Supabase.

## Tecnologias utilizadas

* Python
* Pandas
* Requests
* Difflib
* Unidecode

## Como executar

1. Instalar dependências:

```bash
pip install pandas requests unidecode
```

2. Executar:

```bash
python main.py
```
